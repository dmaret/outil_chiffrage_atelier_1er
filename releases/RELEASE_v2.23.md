# RELEASE v2.23 — Dashboard clinique (Module G)

> Date : 2026-04-30 — Vue d'ensemble globale de tous les inputs cliniques.

## 🎯 Objectif

Proposer un **dashboard global** visualisant l'activité clinique de tous les bénéficiaires en un coup d'œil.

## 🔍 Fonctionnalités principales

- **Bouton 📊 Dashboard** flottant (nouveau FAB avec style terracotta)
- **Métriques globales** :
  - Total inputs tous bénéficiaires
  - Nombre de bénéficiaires avec données cliniques
  - Somme des émotions saisies
- **Résumé par bénéficiaire** : Nom, nombre d'inputs, émotion principale
- **Accès rapide aux stats** : Chaque bénéficiaire a un bouton **📊 Voir stats** pour détails
- **Compteur débriefings** : Total débriefings saisis

## 🎨 Design

- Modale fullscreen-capable (700px max)
- Cartes avec fond blanc/couleurs par bénéficiaire
- Dark mode intégré
