# 📁 Documents - Ressources Centralisées

Ce dossier contient tous les documents, vidéos et fichiers de référence utilisés dans l'application.

## 📚 Structure

- **Fiches techniques**: Descriptions des gestes de travail (REC-01, CTR-01, etc.)
- **Guides**: Manuels de sécurité et procédures (guide-securite.txt)
- **Ressources**: Documents de formation et documents de référence

## 🎥 Types de fichiers

- `.pdf` - Documents PDF (fiches techniques)
- `.mp4` - Vidéos de formation
- `.txt` - Guides texte
- `.doc` / `.docx` - Documents Word
- Autres formats supportés

## 📝 Conventions de nommage

- Fiches gestes: `[CODE]-[description].pdf` (ex: `REC-01-decharger-palette.pdf`)
- Guides: `guide-[nom].txt` (ex: `guide-securite.txt`)
- Vidéos: `[CODE].mp4` (ex: `REC-01.mp4`)

## 🔗 Intégration

Tous les documents sont référencés localement dans `index.html`:
- `./documents/[nom-fichier]`

## 📤 Téléchargement

Les apprenants peuvent télécharger les documents directement depuis l'application Formation tab.
